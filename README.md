# ChatPaper2XMind

论文XMind笔记生成工具——将论文pdf通过ChatGPT转换为带有图片和公式的简要Xmind笔记，提高论文阅读效率。

Paper XMind Notes Generator: Convert PDF to Concise XMind Notes with Images and Formulas Using ChatGPT for Enhanced Reading Efficiency
